## Setup

### Install

- You can clone the repository

```
git clone https://github.com/dangvanthanh/dotfiles.git
```

Then `cd` to `dotfiles` and run `bootstrap`

```
cd dotfiles && source bootstrap.sh
```

### Sensible OS X Defaults

```
./.osx
```
